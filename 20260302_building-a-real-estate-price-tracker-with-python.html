Step 2: Scraping Real Estate Listings

We’ll use web scraping to extract property data from a real estate website. For this example, we’ll use a mock website (e.g., https://example-realestate.com). Replace this with a real URL when deploying.

Example: Scraping Property Listings 

import requests
from bs4 import BeautifulSoup
import pandas as pd
import time
def scrape_listings(url):
    headers = {
        "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/91.0.4472.124 Safari/537.36"
    }
    response = requests.get(url, headers=headers)
    
    if response.status_code != 200:
        print(f"Failed to fetch {url}. Status code: {response.status_code}")
        return pd.DataFrame()
    
    soup = BeautifulSoup(response.text, "html.parser")
    
    # Example: Extract property titles and prices
    listings = []
    for item in soup.select(".property-card"):
        title = item.select_one(".title").get_text(strip=True)
        price = item.select_one(".price").get_text(strip=True)
        price = int(price.replace("$", "").replace(",", ""))
        listings.append({"title": title, "price": price})
    
    return pd.DataFrame(listings)
if __name__ == "__main__":
    url = "https://example-realestate.com/listings"
    df = scrape_listings(url)
    df.to_csv("data/listings.csv", index=False)
    print(f"Saved {len(df)} listings to data/listings.csv")

Key Notes

- User-Agent: Websites often block scrapers by detecting missing headers. Always include a realistic User-Agent. - Error Handling: Check for HTTP errors and handle them gracefully. - Rate Limiting: Add time.sleep(2) between requests to avoid being blocked.

> Tip: Use select() and select_one() with CSS selectors for precise element targeting.

Step 3: Processing and Analyzing the Data

Once we have raw data, we need to clean it and prepare it for analysis.

Clean and Transform Data 

import pandas as pd
def clean_data(df):
    # Remove rows with missing prices
    df = df.dropna(subset=["price"])
    
    # Convert price to numeric
    df["price"] = pd.to_numeric(df["price"], errors="coerce")
    
    # Remove duplicates
    df = df.drop_duplicates(subset=["title"])
    
    # Add a timestamp for tracking
    df["timestamp"] = pd.Timestamp.now()
    
    return df
if __name__ == "__main__":
    df = pd.read_csv("data/listings.csv")
    cleaned_df = clean_data(df)
    cleaned_df.to_csv("data/cleaned_listings.csv", index=False)
    print("Data cleaned and saved to data/cleaned_listings.csv")

Analyzing Trends 

import matplotlib.pyplot as plt
def analyze_data(df):
    # Aggregate average price over time
    df["timestamp"] = pd.to_datetime(df["timestamp"])
    grouped = df.groupby("timestamp")["price"].mean().reset_index()
    
    # Plot the trend
    plt.figure(figsize=(10, 6))
    plt.plot(grouped["timestamp"], grouped["price"], marker="o", linestyle="-")
    plt.title("Average Property Price Over Time")
    plt.xlabel("Date")
    plt.ylabel("Price ($)")
    plt.grid(True)
    plt.tight_layout()
    plt.savefig("data/price_trend.png")
    plt.close()
if __name__ == "__main__":
    df = pd.read_csv("data/cleaned_listings.csv")
    analyze_data(df)
    print("Price trend analysis completed.")

> Best Practice: Store intermediate data (e.g., cleaned_listings.csv) for auditing and debugging.

Step 4: Automating the Tracker

To keep your tracker updated, automate it using cron jobs (Linux/Mac) or Task Scheduler (Windows).

Example: Automating with a Script 

def main():
    url = "https://example-realestate.com/listings"
    df = scrape_listings(url)
    cleaned_df = clean_data(df)
    analyze_data(cleaned_df)
    print("Tracker completed successfully.")
if __name__ == "__main__":
    main()

Scheduling the Script

1. Linux/Mac: Add a cron job with crontab -e: 

   0 * * * * /path/to/venv/bin/python /path/to/src/tracker.py
This runs the script every hour.

2. Windows: Use Task Scheduler to trigger the script at set intervals.

> Warning: Avoid overloading servers with frequent requests. Start with daily updates and adjust as needed.
